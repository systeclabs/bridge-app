## Description

[BRIDGE](https://brige.systec.dev) is an open source platform to work with smart contracts and wallets in the startup ecosystem.

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn run dev
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test

```
